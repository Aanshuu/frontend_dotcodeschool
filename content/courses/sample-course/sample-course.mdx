# Sample Course

This is a sample course to demonstrate the local MDX course structure. It includes examples of sections, lessons, and code files.

## What You'll Learn

- How to structure a course using MDX files
- How to organize sections and lessons
- How to include code examples in your lessons
- How to use the template/solution pattern for exercises

## Prerequisites

- Basic understanding of JavaScript
- Familiarity with Markdown syntax

## Course Structure

This course is organized into sections, each containing multiple lessons. Some lessons include code examples and exercises.
